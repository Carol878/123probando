
# Proyecto-Intermodular-DAW

El proyecto consistirá en la creación de una herramienta de ticketing que proporcione a las empresas una solución integral para la gestión del servicio, todo ello basado en el marco de trabajo ITIL v4.
La implementación de esta herramienta permitirá transicionar a las empresas que busquen
mejorar la eficiencia de sus procesos de un modelo de gestión obsoleto, poco fiable y
desestructurado, principalmente basado en formularios Word y hojas de Excel, a una
aplicación web que les permita organizar, supervisar y optimizar el servicio que ofrecen, a
través de diferentes prácticas.
Así mismo, la herramienta permitirá a la empresa medir el rendimiento a través de
indicadores clave (KPIs), establecer un acuerdo de nivel de servicio (ANS o SLA en inglés)
para asegurar que se cumplan las necesidades del negocio y fomentar la mejora continua
mediante la práctica de gestión del nivel de servicio.

LENGUAJES

Utilizaremos SQL como base de datos, donde almacenaremos todos los registros de
ticketing usuarios y activos. Spring como backend para gestionar flujos de trabajo, seguridad
y exponiendo servicios mediante una API REST y por último Angular para la interfaz web que
consumirá la API para que los usuarios y técnicos interactúen con la herramienta.
